# TechBusiness Website

## Overview
A modern, professional website for a technology business targeting general consumers. The site showcases technology solutions, team profiles, and provides an easy path to conversion.

## Design Philosophy
- **Modern & Professional**: Clean lines, generous white space, and sophisticated color palette
- **User-Centric**: Clear navigation, intuitive layout, and obvious CTAs
- **Trust-Building**: Team profiles, testimonials, and professional imagery
- **Conversion-Focused**: Strategic placement of pricing and contact forms

## Color Palette
- Primary: Blue (#2563eb) - Trust, technology, professionalism
- Secondary: Gray tones - Sophistication and balance
- Accent: White space - Breathing room and luxury feel

## Key Sections

### 1. Navigation
- Sticky header with logo and main navigation
- Clear CTA button for immediate action
- Mobile-responsive with hamburger menu

### 2. Hero Section
- Full-screen impact with professional imagery
- Clear value proposition
- Dual CTAs (primary and secondary actions)
- Modern gradient overlay for text readability

### 3. Services
- 6 core technology services in 3-column grid
- Icon-based visual hierarchy
- Hover effects for engagement
- Clear descriptions of each service

### 4. Team/Testimonials
- Carousel layout showcasing client success stories
- Professional photos and detailed testimonials
- Social proof through star ratings
- Different layout from services for visual variety

### 5. Pricing
- Single clear package at $29/month
- Comprehensive feature list
- Trust indicators (security, instant access, happy clients)
- Stripe integration for easy payment
- Alternative CTA for custom solutions

### 6. Contact Form
- Clean, user-friendly form design
- Contact information prominently displayed
- Visual hierarchy with icons
- Hidden fields for founder_id and project_id
- Success/error messaging

### 7. Footer
- Comprehensive links organized by category
- Contact information repeated
- Social media links
- Professional and complete

## Technical Stack
- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS for utility-first styling
- **Icons**: Lucide React for consistent iconography
- **Forms**: Custom form handling with API integration
- **Images**: Unsplash for high-quality professional imagery

## Key Features
- Fully responsive design (mobile, tablet, desktop)
- Smooth scroll navigation
- Interactive hover states and transitions
- Form validation and submission
- SEO optimized with metadata
- Performance optimized
